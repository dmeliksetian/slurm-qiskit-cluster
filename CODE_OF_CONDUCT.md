# Code of Conduct

As a project within the Qiskit Ecosystem, the **slurm-qiskit-cluster** project adheres to the **[Qiskit Code of Conduct](https://github.com/Qiskit/qiskit/blob/main/CODE_OF_CONDUCT.md)**. 

We are committed to providing a welcoming and inspiring community for all. Participants are expected to adopt these principles and maintain a respectful environment for collaboration.
